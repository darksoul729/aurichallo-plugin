## ğŸšï¸ Controls Explained

### DRIVE (Main Knob - Center)
- **0-30%:** Subtle warmth
- **30-60%:** Moderate saturation
- **60-100%:** Heavy distortion
- **Tip:** Start low, increase to taste

### TONE (Left)
- **0%:** Dark, warm
- **50%:** Neutral (default)
- **100%:** Bright, airy
- **Tip:** Adjust after setting Drive

### MIX (Right Top)
- **0%:** Pure dry (bypass)
- **30%:** Subtle blend (default)
- **100%:** Pure wet
- **Tip:** Lower for parallel processing

### OUTPUT (Right Bottom)
- **0%:** Silent
- **80%:** Default (compensated)
- **100%:** Unity gain
- **Tip:** Match input/output levels

### OVERSAMPLING (Bottom Center)
- **Off:** Lowest CPU
- **2x:** Recommended (default)
- **4x:** Best quality, high CPU
- **Tip:** Use 4x for mastering

### HALO ENGINE (Left Bottom)
- **OFF:** Standard mode
- **ON:** Enhanced saturation + glow effect
- **Tip:** Toggle for more character

## âš¡ Quick Tips

1. **Start Subtle**
   - Default settings are intentionally gentle
   - Increase Drive/Mix gradually

2. **Use Your Ears**
   - Bypass frequently (compare)
   - Don't rely only on meters

3. **Parallel Processing**
   - High Drive + Low Mix = Punch
   - Great for drums and bass

4. **Oversampling**
   - 2x for mixing (default)
   - 4x for mastering
   - Off if CPU is limited

5. **Output Matching**
   - Match input/output levels
   - Avoid "louder = better" trap

6. **Halo Engine**
   - Try toggling ON for more character
   - Visual glow shows intensity

## ğŸ¯ Workflow

### Step 1: Load Plugin
- Default settings load automatically
- Listen to the subtle effect

### Step 2: Adjust Drive
- Start at 20% (default)
- Increase until you hear desired warmth
- Don't overdo it!

### Step 3: Adjust Mix
- Start at 30% (default)
- Increase for more effect

### Step 4: Adjust Tone
- Leave at 50% initially

### Step 5: Match Output
- Watch meters
- Adjust Output to match input level

### Step 6: Fine-tune
- Toggle Halo Engine
- Try different Oversampling
- A/B compare (bypass)
