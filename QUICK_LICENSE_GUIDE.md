## Untuk Developer (Anda)

### Generate License Key untuk Customer

```bash
cd scripts
python3 generate_license_key.py CUSTOMER_HARDWARE_ID
```

Contoh:
```bash
python3 generate_license_key.py A1B2C3D4E5F6G7H8
```

Output:
```
License Key: AH01-A1B2-C3D4-0001-F5E6
```

Kirim key ini ke customer via email.

## Untuk Customer

### Cara Aktivasi License

1. **Buka plugin** di DAW
2. **Klik "Activate License"**
3. **Masukkan license key** yang diterima via email
4. **Klik "Activate"**
5. **Done!** Plugin sekarang licensed

### Cara Dapat Hardware ID

1. Buka plugin
2. Klik "Purchase License"
3. Copy Hardware ID yang ditampilkan
4. Kirim ke support untuk dapat license key

## Troubleshooting

### "Invalid License Key"
- Check typo (copy-paste recommended)
- Pastikan format: XXXX-XXXX-XXXX-XXXX
- Contact support jika masih error

### "License not valid for this computer"
- License tied to specific computer
- Contact support untuk transfer license

### Trial Expired
- Purchase license di website
- Masukkan license key untuk activate
