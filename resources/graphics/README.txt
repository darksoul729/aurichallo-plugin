GRAPHICS FOLDER

Place your installer graphics here:

icon.ico - Application icon (256x256)
installer_banner.bmp - Installer banner (164x314)
installer_icon.bmp - Installer small icon (55x58)

These are optional. If not provided, Inno Setup will use defaults.

To create these:
1. Use any image editor (Photoshop, GIMP, etc.)
2. Export in correct format and size
3. Place in this folder
4. Rebuild installer
