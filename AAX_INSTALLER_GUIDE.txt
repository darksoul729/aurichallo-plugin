ğŸ” CODE SIGNING (For Distribution):

AAX REQUIRES CODE SIGNING:
- Development: Can test unsigned
- Distribution: MUST be signed

SIGNING PROCESS:
1. Get Avid Developer Account ($299/year)
2. Get PACE/iLok account
3. Get code signing certificate
4. Sign with wraptool:
   
   wraptool sign --verbose ^
     --account YOUR_ACCOUNT ^
     --password YOUR_PASSWORD ^
     --wcguid YOUR_GUID ^
     --in AuricHalo.aaxplugin ^
     --out AuricHalo_signed.aaxplugin

5. Use signed version in installer
