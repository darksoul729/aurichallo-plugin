========================================
  AURIC HALO - LICENSE SYSTEM GUIDE
========================================

UNTUK DEVELOPER (KAMU):
-----------------------

1. CARA GENERATE LICENSE KEY:

   Opsi 1 - Pakai Batch Script (Windows):
   - Jalankan: scripts\generate_license_key.bat
   - Copy key yang muncul
   - Kirim ke client

   Opsi 2 - Pakai Python Script (Lebih Advanced):
   - Jalankan: python scripts\generate_license_key.py
   - Pilih method (simple atau advanced)
   - Copy key yang muncul
   - Kirim ke client

2. FORMAT LICENSE KEY:
   
   AURIC-XXXX-XXXX-XXXX-XXXX
   
   Contoh:
   - AURIC-A1B2-C3D4-E5F6-G7H8
   - AURIC-1234-5678-9ABC-DEF0

3. VALIDASI KEY:
   
   Saat ini validasi sederhana:
   - Key harus dimulai dengan "AURIC-"
   - Minimal 16 karakter
   
   Untuk production, kamu bisa upgrade ke:
   - Validasi dengan server online
   - Cryptographic signature
   - Hardware ID binding

4. CARA KERJA:
   
   a. Client install plugin
   b. Plugin jalan dalam trial mode (30 hari)
   c. Setelah 30 hari atau client mau aktivasi:
      - Buka plugin
      - Klik tombol "License" atau "Activate"
      - Input license key
      - Klik "Activate"
   d. Plugin tersimpan di:
      %APPDATA%\Auric Audio\Auric Halo\license.dat

5. DEACTIVATE LICENSE:
   
   Kalau client mau pindah komputer:
   - Hapus file: %APPDATA%\Auric Audio\Auric Halo\license.dat
   - Atau pakai tombol "Deactivate" di plugin

========================================

UNTUK CLIENT:
-------------

1. TRIAL MODE:
   
   - Plugin bisa dipakai GRATIS selama 30 hari
   - Tidak perlu license key
   - Semua fitur tersedia

2. CARA AKTIVASI:
   
   a. Buka plugin di DAW kamu
   b. Klik tombol "License" atau "Activate"
   c. Input license key yang diberikan developer
   d. Klik "Activate"
   e. Selesai! Plugin sekarang licensed

3. FORMAT LICENSE KEY:
   
   AURIC-XXXX-XXXX-XXXX-XXXX
   
   Contoh: AURIC-A1B2-C3D4-E5F6-G7H8

4. TROUBLESHOOTING:
   
   Q: License key tidak diterima?
   A: Pastikan format benar (AURIC-XXXX-XXXX-XXXX-XXXX)
   
   Q: Mau pindah komputer?
   A: Deactivate di komputer lama, lalu activate di komputer baru
   
   Q: Trial sudah habis?
   A: Hubungi developer untuk beli license key

========================================

UPGRADE SISTEM LISENSI (OPSIONAL):
-----------------------------------

Untuk production yang lebih secure, kamu bisa upgrade:

1. ONLINE VALIDATION:
   - Buat server API untuk validasi key
   - Key dicek ke server setiap kali plugin dibuka
   - Bisa track berapa kali key dipakai

2. HARDWARE ID BINDING:
   - Key terikat ke hardware ID komputer
   - Tidak bisa dipakai di komputer lain
   - Lebih secure tapi kurang fleksibel

3. CRYPTOGRAPHIC SIGNATURE:
   - Key di-sign dengan private key
   - Plugin verify dengan public key
   - Tidak bisa di-fake

4. SUBSCRIPTION MODEL:
   - Key expire setelah periode tertentu
   - Client harus renew subscription
   - Recurring revenue

Untuk implementasi ini, butuh:
- Backend server (Node.js, PHP, Python)
- Database (MySQL, PostgreSQL)
- API endpoint
- Update code di LicenseManager.cpp
