PARAMETERS:

TONE (0-100)
- Controls the tonal character
- Lower values: darker, warmer
- Higher values: brighter, more presence

DRIVE (0-100)
- Controls saturation amount
- Lower values: subtle enhancement
- Higher values: aggressive saturation

MIX (0-100%)
- Blends dry and wet signal
- 0%: Dry signal only
- 100%: Fully processed

OUTPUT (0-100)
- Controls output level
- Use to compensate for gain changes
- Default: 50

HALO ENGINE (ON/OFF)
- Visual effect toggle
- Does not affect audio

TROUBLESHOOTING:

Plugin doesn't appear in DAW:
- Rescan plugins in your DAW
- Check plugin is in correct folder
- Restart DAW

No sound:
- Check MIX is not at 0%
- Check OUTPUT level
- Check plugin is not bypassed

Distortion/Clipping:
- Reduce DRIVE amount
- Lower OUTPUT level
- Check input level
